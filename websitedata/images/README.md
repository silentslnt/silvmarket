# 🖼️ Silv Market — Images Folder

Drop your images here, then reference them in index.html using:
  images/filename.png

## Files to add:
| Filename          | Used for                        |
|-------------------|---------------------------------|
| logo.png          | Nav logo + browser tab icon     |
| banner.png        | Hero section background         |
| bf-dragon.png     | Blox Fruits Dragon card image   |
| bf-kitsune.png    | Kitsune card image              |
| (add more...)     |                                 |

## How to add an image:
1. Drag the image file into this folder in VS Code
2. git add . → git commit -m "add images" → git push
3. It will be live at: https://silvmarket.shop/images/filename.png
